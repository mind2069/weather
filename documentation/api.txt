🧠 How the Data Comes Back (Important Concept)

Open-Meteo returns parallel arrays:

{
  "hourly": {
    "time": [
      "2026-04-05T00:00",
      "2026-04-05T01:00"
    ],
    "temperature_2m": [21.3, 20.9],
    "weather_code": [1, 2],
    "wind_speed_10m": [8.2, 7.9]
  }
}

Index = the hour.

So:

hourly.time[i]
hourly.temperature_2m[i]
hourly.wind_speed_10m[i]

belong together.

⭐ Example Next.js Normalization
function groupByDay(hourly) {
  const days = {}

  hourly.time.forEach((t, i) => {
    const day = t.split("T")[0]

    if (!days[day]) days[day] = []

    days[day].push({
      time: t,
      temp: hourly.temperature_2m[i],
      wind: hourly.wind_speed_10m[i],
      uv: hourly.uv_index[i],
      code: hourly.weather_code[i]
    })
  })

  return days
}

This is the key step most weather apps miss.
